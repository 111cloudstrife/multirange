multirange

A tiny polyfill for HTML5 multi-handle sliders.

<input type="range" multiple value="10,80" />

Features

  • Only 0.5KB minified & gzipped!
  • Keyboard accessible
  • Clicking on the track still works
  • Handles can be moved past each other
  • Customizable styling
  • JS properties polyfilled too (input.value, input.valueLow, input.valueHigh)

Limitations

  • Requires support for <input type="range"> (duh).
  • Fully works only in browsers supporting CSS Variables. Fallback in others.
  • Requires Bliss.
